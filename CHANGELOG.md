# Changelog

## v1.6.0 (2026-02-22)

### Updates
- **Update modal** — Moved to Dashboard as a blocking popup with black/white theme
- **Professional copy** — Updated modal title and description
- **Download flow** — In-app download with progress; "Download manually" fallback only when needed

### When update is available
- Voice overlay (Ctrl+Win) and Composer (Alt+Win) show error until you update
- Rephrase shortcut (Ctrl+Shift+Space) shows error until you update

### Installer
- Desktop shortcut option
- Start Rephrase at Windows login (Startup folder shortcut)
- Settings toggle to enable/disable launch at startup

### Fixes
- Start-with-Windows uses Startup folder shortcut for reliable behavior

### Removed
- Hands-free mode shortcut (removed from Settings)

## v1.0.0 (2026-02-22)
